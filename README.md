# Stock Price Prediction (Time Series)

This project uses a Linear Regression model to predict the daily closing price of Apple (AAPL) stock based on historical data.

## Features
- Fetches real-time stock data using `yfinance`.
- Implements time-series forecasting by shifting target variables.
- Visualizes actual vs. predicted prices using `matplotlib`.
